# Contributing to WaraNode

First off, thank you for considering contributing to Wara! It's people like you that make Wara such a great tool.

## ⚠️ Important Disclaimer

**Wara is a decentralized P2P protocol.**

*   **Responsibility**: By running a WaraNode, you are acting as an independent host. The developers of the Wara protocol do not control, oversee, or host the content shared between peers.
*   **Compliance**: Contributors must ensure that any code or features proposed do not violate local laws.
*   **Liability**: The software is provided "as is", without warranty of any kind. Use it at your own risk.

## How Can I Contribute?

### Reporting Bugs
*   Check the issues page to see if the bug has already been reported.
*   If not, open a new issue. Clearly describe the problem and include steps to reproduce the error.

### Suggesting Enhancements
*   Open an issue with the tag "enhancement".
*   Describe the feature you'd like to see and why it would be useful.

### Pull Requests
1.  Fork the repo and create your branch from `master`.
2.  If you've added code that should be tested, add tests.
3.  If you've changed APIs, update the documentation.
4.  Ensure the test suite passes.
5.  Check your spelling and grammar.
6.  Submit a Pull Request.

## Community Guidelines
*   Be respectful and inclusive.
*   Focus on technical excellence and decentralization.
*   Protect user privacy at all costs.
